# Promptware OS Kernel Suite

![OS Version](https://img.shields.io/badge/OS-v1.2-blue) ![Kernels](https://img.shields.io/badge/Kernels-5_Available-green) ![License](https://img.shields.io/badge/license-MIT-purple)

> **"Truth > Speed."**

This repository contains the reference implementation of **Promptware OS v1.2**, a standard for installing cognitive operating systems into Large Language Models (LLMs).

It hosts the **Promptware Library**, a collection of 5 distinct neurosymbolic kernels that transform the LLM into a specialized agent.
