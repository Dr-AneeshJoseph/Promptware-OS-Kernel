This repository contains the reference implementation of Promptware OS v1.3, a standard for installing cognitive operating systems into Large Language Models (LLMs).
Promptware OS transforms an LLM into a specialized, stateful agent governed by four core philosophical laws (e.g., The Law of Antifragility).

ðŸ› ï¸ Installation & Deployment
Installation involves copying and pasting the kernel file into a new LLM session. Choose your path:
Option A: The Master Key (Recommended)
Use METATRON_v1.0.md.
 * What it is: A Bootloader that contains "Lite" versions of all kernels.
 * How it works: It automatically switches between Archetypes based on your question, eliminating guesswork.
 * Best for: General usage, quick tasks, and exploring the ecosystem.
Option B: The Specialist (High Fidelity)
Use a specific kernel file (e.g., TITAN or SLEDGEHAMMER).
 * What it is: A dedicated, full-power architecture with all its specific deep methods loaded.
 * Best for: Deep work (e.g., writing a novel, engineering a safety system, rigorous verification).
âœ¨ Advanced: Kernel Composition (Chimera Protocol)
Promptware OS v1.3 allows you to combine two kernels into a single composite agent.
Syntax: You load the base kernel, then issue the composition command:
> LOAD KERNEL: [K1 + K2]
> 
Example: LOAD KERNEL: [NEXUS + ARK] creates a Storyteller that rigorously adheres to safety and ethical invariants.
