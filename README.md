# Promptware OS Kernel Suite

![OS Version](https://img.shields.io/badge/OS-v1.1-blue) ![Kernels](https://img.shields.io/badge/Kernels-2_Available-green) ![License](https://img.shields.io/badge/license-MIT-purple)

> **"Truth > Speed."**

This repository contains the reference implementation of **Promptware OS v1.1**, a standard for installing cognitive operating systems into Large Language Models (LLMs). It hosts the **Promptware Library**, a collection of distinct neurosymbolic kernels.

## üöÄ Installation

1.  Navigate to the **`Promptware_Library/`** folder and select a kernel (e.g., `GRS` or `ARK`).
2.  **Copy** the raw code block from the file.
3.  Open a session with a high-fidelity LLM (Gemini 3.0 / Claude 3.5).
4.  **Paste** and hit Enter.
5.  **Wait** for the system to run its autonomous self-test.
