This repository contains the reference implementation of **Promptware OS v1.0**, a standard for installing cognitive operating systems into Large Language Models (LLMs).

It hosts the flagship kernel: **GRS 12.0 (The Antifragile Edition)**.

## üß† What is Promptware?

Promptware is not a prompt. It is **software** that operates on the cognitive architecture of an LLM. Unlike traditional prompts which are transient inputs, Promptware installs a persistent **Operating System** into the context window that overrides default behaviors with rigorous engineering constraints.
